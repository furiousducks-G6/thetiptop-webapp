Besoin d'aide ?

1. Comment participer au jeu-concours ?

Pour participer, il vous suffit d'entrer le code que vous trouverez sur votre ticket de caisse dans la section "Participer" de notre site web. Assurez-vous de créer un compte ou de vous connecter avec Google/Facebook.

2. Que faire si mon code ne fonctionne pas ?

Si votre code ne fonctionne pas, vérifiez qu'il a été correctement saisi. Si le problème persiste, contactez notre service client en fournissant une photo claire de votre ticket de caisse.

3. Comment réclamer mon lot ?

Une fois votre code validé, vous pouvez réclamer votre lot en vous rendant en boutique ou en suivant les instructions pour une livraison à domicile. Consultez la section "Réclamer mon lot" sur notre site pour plus de détails.

4. Comment nous contacter ?

Vous pouvez nous contacter via le formulaire de contact sur notre site ou directement à l'adresse email {{'contact@thetiptop.com'}}.
